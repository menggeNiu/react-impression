### Examples

**basic**

```js
<Form>
  <Form.Group>
    <label>Radio</label>
    <Radio ref="radio" value={1} onChange={this.onChangeHandle}>
      single
    </Radio>
  </Form.Group>
</Form>
```

**disabled**

```js
<Form>
  <Form.Group>
    <label>Disabled</label>
    <Radio disabled>Yes</Radio>
  </Form.Group>
</Form>
```
